# Case

3D-printable enclosure for the K-Pop Digital Photocard device.

## Parts

- `front.stl` — Front shell with display cutout
- `back.stl` — Back cover
- `button.stl` — Boot button cap

## Assembly

1. Print all three parts
2. Heat-set four M3x4mm threaded inserts into the front part using a soldering iron
3. Place the ESP32-S3 board into the front shell and snap in the button cap
4. Connect the lipo battery to the board
5. Use double-sided tape to secure the battery to the inside of the back cover
6. Place the back cover on and fasten with four M3x12mm screws
